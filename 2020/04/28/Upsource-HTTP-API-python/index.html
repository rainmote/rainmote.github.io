背景

Upsource是一个非常出色的CodeReview工具，在和其他系统联动时需要通过API进行访问，于是使用python对其API进行封装，过程中一步一步思考如何使用python装饰器简化封装，达到十分简洁的封装。
本文主要介绍利用动态生成函数和装饰器两种方法简化API封装的实现。
